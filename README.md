# NanoMatch - Ultra Low Latency Order Matching Engine
A high-performance, low-latency Limit Order Book implementation designed for sub-microsecond execution.

## Key Features
* Price-Time Priority Matching
* Limit & Market Order Support
* Thread-safe Design
* Performance Benchmarking (Upcoming)

## Tech Stack
* C++17/20
* CMake

## Build Instructions
```bash
mkdir build && cd build
cmake ..
make -j4
